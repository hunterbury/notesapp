# Notes App


This is a single-page app that uses React for the front-end and Django Rest Framework for the back-end.
## Tech Stack

**Client:** React, CSS, HTML

**Server:** Django, REST API


## Features

- React front-end
- CRUD functionality
- REST API
- Django models, serializers
